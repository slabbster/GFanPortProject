Gfan binaries

Disclaimer: The author has no chance of knowing that these files are virus free. Use them at your own risk.
For your convenience Gfan has been precompiled with gmp and cddlib for selected systems.

Gmp and cddlib are licensed under the GNU LGPL and GNU GPL respectively.

The precompiled executables make the installation process easy:
  • Download the executable file for your system to the directory where you want Gfan to be.
  • Change the name of the executable file to gfan.
  • Open a terminal/shell and go to the directory to where you put the gfan executable file.
  • Make sure that the executable file is executable by doing chmod 755 gfan in your shell.
  • Run ./gfan installlinks to create symbolic links.
  • For convenience set up your path by adding the line
    set PATH=$PATH:/Users/yourusername/thedirectoryname
    to either the file .profile or .bash_profile depending on your system.
    Both files are located in your home directory. You should change the above line,
    so that it points to the directory where you put the executable file.
